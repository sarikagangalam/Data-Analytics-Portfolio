# Product Sub-Category Performance Analysis

## Project Overview
This project analyzes product sub-category performance using sales, quantity, profit, and discount metrics to identify high-performing and loss-making categories.

## Tools Used
- Excel
- Power BI

## Key Insights
- Some sub-categories generate high sales but low or negative profit.
- Tables and Bookcases are loss-making categories.
- Copiers generate high profit despite lower sales volume.

## Business Value
Helps businesses optimize pricing, discount strategies, and focus on profitable product segments.
